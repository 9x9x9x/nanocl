NAME
====

nanocl-revert - Revert a configuration file

SYNOPSIS
========

**nanocl-revert** \[**-h**\|**\--help**\] \<**-f **\>

DESCRIPTION
===========

Revert a configuration file

OPTIONS
=======

**-h**, **\--help**

:   Print help information

**-f**=*FILE\_PATH*

:   .yml conf file to revert
